My First Blog!

This week I read an article explaining modern JavaScript. I have known how to use HTML and CSS
but never really had a good grasp on JS. This article explained to me how HTML and JavaScript
can interact with one another to improve the quality (and readability) of a website.
I learned that you can include JS scripts in HTML. This is shown in the following code:

    <!-- index.html -->
    <!DOCTYPE html>
    <html lang="en">
    <head>
        <meta charset="UTF-8">
        <title>JavaScript Example</title>
        <script src="index.js"></script>
    </head>
    <body>
        <h1>Hello from HTML!</h1>
    >/body<
    </html>
We can link a JS script with the script keyword in HTML and then call functions from that script's library.
It is also interesting that you can use functions from JavaScript in the order they are added to the
HTML file. So if I add a JS file before another one, I can call the first scripts methods in the second script.

Also, everytime a library is updated you would have to search and update it as well as your code . Package
