This week I finished reading the article on JavaScript that I've been reading these past 3 weeks! This section
was on transpiling code which is converting code from one language to another. Nowadays, the transiplers for
JavaScript are TypeScript or babel. I think babel is a cool name because it is a play on the word "babble"
which means talk that is incomprehensible. Obviously while to us JavaScript is understandable to others
it may honestly be incomprehensible which is kind of funny. To install babel as an npm package (which
I wrote about in my last blog) you just have to run the following command:


$ npm install @babel/core @babel/preset-env babel-loader --save-dev

After running this line and editing your .config file, you can begin to translate your code. In the article
they convert ES2015 into JavaScript. They converted this line in ES2015:

console.log(`Hello ${name}, how are you ${time}?`);

into this line in JavaScript:

console.log('Hello ' + name + ', how are you ' + time + '?');

This is extremely useful as translating code can take an extreme amount of work and time but this
problem is solved thanks to transpilers. This was the last section of this article and unfortunately
it was quite short. However, I did enjoy reading this section and thought it was worth including
in my blog. The ability to quickly translate code allows for so much flexibility with development.
I learned a lot from this article and this has definitely helped me to become more versed in JavaScript
development. I think the next thing I will look into will be CSS as it is somethign that I am still not
and expert in but I would like to be.
